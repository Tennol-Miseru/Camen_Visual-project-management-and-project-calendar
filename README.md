# Camen_Visual-project-management-and-project-calendar
一个工程日历小工具， 主要功能为可视化的日期事项条与项目工序页。 支持导入导出数据来实现跨端同步。

A small engineering calendar tool whose core features include visualized timeline bars for date-based tasks and dedicated pages for project workflows. It also supports data import and export to enable cross-device synchronization. 

主要用于解决我的多项目并行时思路不清、难以用便签或脑内统筹的问题。
整个工具几乎全部使用Codex完成，如果你有类似的问题，欢迎拿来自用和修改。

Primarily designed to address the lack of clarity when managing multiple projects in parallel, especially when it’s difficult to organize tasks using notes or mental tracking.
The tool was developed almost entirely with Codex, and if you have similar challenges, feel free to use and modify it.
